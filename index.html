Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan gaya visual (misalnya, warna, font, tata letak) dari halaman web. JavaScript digunakan untuk menambahkan interaksi, fungsi dan dinamika ke halaman web. CSS dan JavaScript bekerja sama untuk membuat pengalaman pengguna yang lebih menarik dan interaktif di web.

CSS (Cascading Style Sheets)
Fungsi Utama: Mengatur tampilan dan desain halaman web.

  • Memisahkan konten dari desain: HTML untuk struktur/konten, CSS untuk tampilan.
  • Mengatur tampilan secara konsisten dari satu file CSS.

JavaScript
Fungsi Utama: Menambahkan interaktivitas dan logika dinamis pada halaman web.

  • Membuat halaman interaktif: tombol, dropdown, slideshow.
  • Validasi form sebelum dikirim ke server.

Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

CSS ( Cascading Style Sheets) dan Javascript dapat diintegrasikan dalam pemrograman web untuk menambahkan gaya dan interaktivitas pada halaman. HTML, CSS dan Javascript bekerja bersama untuk menciptakan pengalaman web yang dinamis dan menarik.

  • Inline CSS:
    <p style="color: blue; font-size: 18px;">Teks ini berwarna biru</p>
  • Internal CSS:
    <style> p { color: blue; } </style>
  • External CSS:
    <link rel="stylesheet" href="style.css">
  • Inline JS:
    <button onclick="alert('Halo!')">Klik Aku</button>
  • Internal JS:
    <script> function halo(){ alert("Halo!"); } </script>
  • External JS:
    <script src="script.js"></script>

Jelaskan apa saja keuntungan menggunakan JavaScript dan CSS?

  • JavaScript dan CSS memiliki keuntungan yang berbeda namun saling melengkapi dalam pengembangan web. JavaScript memberikan interaktivitas dan dinamitas pada halaman web, memungkinkan manipulasi konten, validasi formulir , dan bahkan pengembangan aplikasi web kompleks. CSS disisi lain, bertanggung jawab atas terampilan dan presentasi visual, memungkinkan pengembangan untuk mengontrol warna textual style, tata letak, dan animasi meningkat pengalaman pengguna.

Jelaskan apa saja keuntungan menggunakan Bootstrap?

  • Bootstrap menawarkan banyak keuntungan dalam pengembangan web, terutama dalam hal kemudahan, kecepatan dan responsivitas.
  • Responsif: otomatis menyesuaikan ukuran layar.
  • Komponen siap pakai: tombol, navbar, modal.
  • Cepat dan efisien: hemat waktu.
  • Kompatibel di semua browser.
  • Dokumentasi lengkap dan komunitas besar.
  • Mudah dikustomisasi dan diintegrasikan dengan jQuery/React.

Contoh penggunaan Bootstrap dan penjelasannya

Contoh kode Bootstrap tombol:

<button class="btn btn-primary">Klik Saya</button>

Penjelasan:

  • btn: class dasar tombol Bootstrap.
  • btn-primary: memberi warna biru Bootstrap.
  • container, text-center, mt-5: digunakan untuk mengatur layout dan posisi konten agar lebih rapi dan berada di tengah.
